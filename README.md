# silberschatz
mDBMS

## Usage

### Server

```bash
uv run src/server.py
```

### Client

```bash
uv run src/client.py
```
## Seed

```bash
uv run scripts/seed_sample_data.py
```

## Testing Example
```bash
uv run python -m unittest discover -s tests/optimizer
```
